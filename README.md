## 特殊矩阵的压缩存储

|            |                           元素下标                           |             表长             |
| :--------: | :----------------------------------------------------------: | :--------------------------: |
|  对称矩阵  | i(i-1)/2 + j -1 （下三角和主对角）  j(j-1)/2 + i -1   (上三角) |           n(n+1)/2           |
|  三角矩阵  |    i(i-1)/2 + j -1  （下三角和主对角） n(n+1)/2（上三角）    |          n(n+1)/2+1          |
| 三对角矩阵 |                          k= 2i +j-3                          | i= (k+ 1)/3 + 1，j=k -2i + 3 |
|  稀疏矩阵  |                存储方法：  十字链表法，三元组                |                              |
